Add\+Customer krijgt Sign\+Up\+Request\+Customer binnen en Sign\+Up\+Request\+Customer\+Private. Elk account heeft als algemene registratie een email en accounttype. 

Bij een particulier (Private) account geeft de gebruiker alleen een email en een accounttype door. Bij een zakelijk (Business) account geeft de gebruiker een email, accounttype en KvK nummer door (ID\textquotesingle{}s worden ingevuld door de database)

Bij een particulier account wordt nog een tweede query uitgevoerd (zie Add\+Private\+Customer\+Details). Als deze query mocht falen wordt de gehele transactie gerolledbacked.
